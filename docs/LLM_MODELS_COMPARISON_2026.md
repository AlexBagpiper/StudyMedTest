## 2. Глубокий анализ кандидатов

### DeepSeek (Китай) - "Убийца затрат"
*   **Плюсы**: Невероятно низкая цена. По тестам 2025-2026 года превосходит GPT-4 во многих медицинских задачах.
*   **Минусы**: Иногда нестабильное время ответа (latency) в часы пик.
*   **Вердикт**: Рекомендуется как основная модель для массовой проверки ответов студентов.

### Qwen (Alibaba) - "Золотой стандарт для RU"
*   **Плюсы**: Огромная база обучения на мультиязычных данных. Прекрасно понимает русскую медицинскую терминологию.
*   **Минусы**: Требует аккаунта Alibaba Cloud.
*   **Вердикт**: Лучшая альтернатива DeepSeek, если нужна большая стабильность.

### GigaChat (Сбер) - "Безопасность и Право"
*   **Плюсы**: Работает внутри РФ, не требует VPN/прокси, оплата в рублях по счету.
*   **Минусы**: В 7-10 раз дороже китайских аналогов. Модель Lite может "галлюцинировать" на сложных терминах.
*   **Вердикт**: Использовать только если есть жесткое требование хранить/обрабатывать всё строго в РФ.

## 3. Рекомендация по интеграции в StudyMedTest

Для максимальной экономии (в 15-20 раз относительно текущего YandexGPT) рекомендую следующую схему:
1. **Основная модель**: `DeepSeek-V3` (через OpenAI SDK).
2. **Резервная модель (Fallback)**: `Qwen-Plus` или `GPT-4o-mini`.
3. **Локальная модель**: `Llama 3.3 70B` (если есть свои сервера с GPU) — затраты только на электричество.

**Экономический эффект**: Переход с YandexGPT на DeepSeek снизит стоимость проверки одного теста с ~5-10 рублей до ~0.3-0.5 рублей.
